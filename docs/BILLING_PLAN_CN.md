# 收费系统与定价策略（中国市场）

> 适用场景：KidneySphere AI 随访登记系统（团队席位、研究协作、多中心项目）

## 1. 当前建议定价（可直接上线）

### Pro 团队版（小团队）
- ¥999 / 月，含 5 席位
- 超出席位：¥99 / 席位 / 月
- 年付：¥9,590 / 年（8 折）

### Institution 机构版（多中心）
- ¥2,999 / 月，含 20 席位
- 超出席位：¥79 / 席位 / 月
- 年付：¥28,790 / 年（8 折）

### Enterprise（定制）
- ¥6,999 / 月起
- 可包含：专属支持、定制报表、合规条款协作
- 年付默认 8 折或按合同约定

---

## 2. 退款策略（“随时可退款”可持续版本）

为了同时满足“随时可退款”和平台可持续经营，建议采用：

- 月付：随时取消，按剩余未使用天数比例退款（prorated）
- 年付：随时取消，按剩余未使用天数比例退款；已使用期间按月价折算
- 退款到账时效：3-7 个工作日原路退回

### 风险控制补充
- 限制频繁开通-退款行为（例如 30 天内超过阈值进入人工审核）
- 首次大额年付支持 7 天无理由全额退款（可选）
- 全量保留退款审计日志

---

## 3. 产品包装建议（提升成交率）

### 套餐页面必写三点
1. 包含席位数（降低决策成本）
2. 超席位单价（透明可预期）
3. 年付节省金额（把“8 折”转成具体省多少钱）

### 推荐展示方式
- 月付价为主，年付价显示“立省 X 元”
- 在团队版卡片加“适合课题组/科室”
- 在机构版卡片加“适合多中心协作”

---

## 4. 与当前系统的映射

当前数据库已有 `trial / pro / institution` 订阅层级，可直接映射：

- Pro 团队版 -> `subscription_plan = 'pro'`
- Institution 机构版 -> `subscription_plan = 'institution'`
- 试用态 -> `subscription_plan = 'trial'`

并通过 `subscription_active_until` 管理有效期。

---

## 5. 4~6 周观察指标（未知交易规模时的最优做法）

上线后建议每周看以下指标：

- 访客到付费转化率
- 每个付费项目平均席位数
- 年付占比
- 退款率（整体/年付）
- 扣费成功率

### 调整规则（简版）
- 转化率高 + 退款率低：可尝试提价 10%~15%
- 转化率低 + 退款率高：保持月价，优化年付退款条款与套餐说明
- 席位利用率持续 >80%：提高超席位单价或上调基础包价格

---

## 6. 条款与沟通文案建议

建议在购买页与协议中明确：

- 平台为研究管理工具，不提供临床诊疗建议
- 不得录入可识别个人身份信息（PII）
- 退款规则、周期与到账时效
- 席位定义（活跃成员/管理员是否计入）
- 订阅取消后的数据读取权限说明
